## 1. 项目概述

### 1.1 项目背景

《常州年轻人自救指南》是一款面向常州本地年轻人的相亲交友小程序，致力于通过技术手段解决年轻人社交难、脱单难的问题。项目名称带有自嘲和幽默色彩，旨在打造一个真实、可信、有趣的本地化交友平台。

### 1.2 核心定位

- **常州本地化**：用户常州本地化居多
- **真实可信**：多重认证机制，确保用户信息真实
- **轻量级匹配**：简洁高效的匹配流程，降低用户使用门槛

### 2.2 技术选型清单

| 组件     | 技术选型          | 版本           | 主要用途                         |
| :------- | :---------------- |:-------------| :------------------------------- |
| 后端核心 | Spring Boot       | 3.2.7        | 应用基础框架                     |
| ORM框架  | MyBatis-Plus      | 3.5.6        | 数据访问层，增强CRUD             |
